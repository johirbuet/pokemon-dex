# Pokémon Dex

A small React + Vite app to search Pokémon using the PokéAPI.

Live deployment: https://pokemon-dex-five-coral.vercel.app/

## Run locally

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Notes

- Requires Node.js (16+ recommended).
- The app uses Vite and `@vitejs/plugin-react`.
